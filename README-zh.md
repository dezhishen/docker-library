# docker-library

[English](README.md) | [简体中文](README-zh.md)

## 项目简介

本项目由 OpenListTeam 维护，提供了用于构建 OpenList 项目相关镜像的**基础镜像**。旨在帮助开发者和使用者降低镜像构建复杂度，提高构建效率，减少重复工作。

## 如何自行构建镜像

如需自行构建镜像，请按照以下步骤操作：

1. **Fork** 本仓库到您的 GitHub 账号。
2. **配置变量和密钥**：  
   在您 Fork 后的仓库的 Actions 设置页，添加所需的变量和密钥。具体请参考下方[变量与密钥](#变量与密钥)一节。
3. **运行构建工作流**：  
   进入仓库的 “Actions” 标签页，手动触发相关的工作流以构建镜像。

## 变量与密钥

| 名称                  | 类型     | 说明                                                  | 是否必填 | 默认值               |
|---------------------|--------|-----------------------------------------------------|------|-----------------|
| `IMAGE_NAME`         | 变量     | 构建的 Docker 镜像名称                                 | 否   | `openlist-base-image`     |
| `DOCKERHUB_ORG_NAME` | 变量     | 推送镜像到 Docker Hub 时使用的组织或用户名               | 否   | 仓库所有者           |
| `GHCR_ORG_NAME`      | 变量     | 推送镜像到 GitHub Container Registry 时的组织或用户名     | 否   | 仓库所有者           |
| `DOCKERHUB_TOKEN`    | 密钥     | 推送镜像到 Docker Hub 时用于认证的 Token                | 是   | 无                  |

**说明：**  
如果未指定上述变量，Docker Hub 和 GHCR 相关变量将默认使用仓库所有者名称。

## 其他说明

- 带有 `aria2` 和 `aio` 标签的镜像即将弃用，请尽快迁移至专用镜像。
